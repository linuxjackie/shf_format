# 貢獻指南
感謝你對本項目的興趣！以下是如何貢獻的一些指南：
- **Fork這個項目**
- 創建你的功能分支 (`git checkout -b feature/AmazingFeature`)
- 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
- 推送到分支 (`git push origin feature/AmazingFeature`)
- 創建一個新的Pull Request
## 代碼風格
請遵循以下代碼風格：
- 使用Python的PEP 8標準。
- 為所有公共方法和類添加文檔字符串。
- 確保代碼通過所有測試。
## 提交消息
請確保你的提交消息清晰且描述性強。使用以下格式：
[類型] 描述
示例:
[修正] 修復了SHF解析中的錯誤
[功能] 增加了批量處理功能
